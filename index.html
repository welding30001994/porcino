ERROR DE PERMISOS EN FIREBASE

Necesitas configurar las reglas de seguridad en Firebase Console

{
  "rules": {
    ".read": true,
    ".write": true,
    "signals": { ".read": true, ".write": true },
    "chat": { ".read": true, ".write": true },
    "posts": { ".read": true, ".write": true },
    "users": { ".read": true, ".write": true }
  }
}

Ve a Firebase Console > Realtime Database > Reglas y pega estas reglas
